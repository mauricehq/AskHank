Design Rules
  • NO Emojis. Anywhere. Hank doesn't decorate.
  • NO Exclamation marks in UI copy. Hank doesn't shout.
  • NO Rounded, bubbly, playful patterns. This isn't a wellness app.
  • NO Gradient backgrounds or flashy effects.
  • YES Dry, direct copy. Short sentences. No filler.
  • YES Burnt orange as the only accent. One color, used consistently.
  • YES Plenty of whitespace. Let the words breathe.
  • YES DM Sans + DM Mono only. No additional typefaces.
